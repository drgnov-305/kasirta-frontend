## Tentang KasirTa'

KasirTa' adalah aplikasi Point of Sale (POS) berbasis web yang dirancang khusus untuk membantu pelaku UMKM kuliner — warung makan, kafe, dan kantin kampus — dalam mengelola transaksi harian secara digital. Dikembangkan sebagai bagian dari program pengabdian masyarakat Kelompok 11, Universitas Ciputra Makassar.

---

## Fitur Utama

- 🧾 **Kasir Digital** — Proses transaksi cepat dengan kalkulasi otomatis
- 📊 **Laporan Real-Time** — Pantau pendapatan dan histori transaksi harian
- 📦 **Manajemen Stok** — Notifikasi otomatis saat stok menipis atau habis
- 🖨️ **Printer Thermal** — Cetak struk otomatis setelah transaksi selesai
- 💬 **Notifikasi WhatsApp** — Laporan harian langsung ke WhatsApp pemilik
- 📡 **Mode Offline** — Transaksi tetap berjalan tanpa koneksi internet
